
# Test project CFT SHIFT 2024 winter

![CFT](https://team.cft.ru/sites/job/SectionImages/cft_shift.png)

## The short description of app
The application makes a request and receives data about random people from the site https://randomuser.me /. It displays brief information about them, and when you click on one of the people, the application takes you to the screen with full information about the users. In this screen, you can also click on the dialer, mail and coordinates button to send the user to the application to process this data.
